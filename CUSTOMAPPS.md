# Custom Apps

## Official Apps

### new-releases

Add a page showing new releases from artists you follow.

### reddit

Add a page showing popular music on certain music subreddits.

### lyrics-plus

Add a page with pretty scrolling lyrics.

### marketplace

Add a page where you can browse extensions, themes, apps, and snippets. Using the marketplace does not work with this flake, however it is still here in order to allow for browsing.

## Community Apps

### localFiles

Add a shortcut to see just your local files.

### nameThatTune

Heardle.app for spicetify. [Source](https://github.com/theRealPadster/name-that-tune)

### betterLibrary

Adds a library page

### library

Adds a library page. [Source](https://github.com/harbassan/spicetify-apps)

### statistics

Adds a page where you can see an analysis of your listening history. [Source](https://github.com/harbassan/spicetify-apps)

### better-local-files 

Adds a page where you can browse your local files. [Source](https://github.com/Pithaya/spicetify-apps)

### history-in-sidebar 

Adds a shortcut to your recently listened page. [Source](https://github.com/Bergbok/Spicetify-Creations)

### playlist-tags 

Allows you to search playlists using tags. [Source](https://github.com/Bergbok/Spicetify-Creations)
